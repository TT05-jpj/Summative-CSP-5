How to use the Caretaker Dashboard

Add an object — Click the yellow Add Object + card to open the scanner. Point the camera at an object, tap Scan Object, then save it with a name and optional instructions.

View Instructions — See exactly what the user will read when they scan that object.

Edit Instructions — Update the name or instructions for an existing object.

Delete — Permanently remove an object from the list.

Languages — Use the dropdowns at the top to set the caretaker display language and the language the user will see.

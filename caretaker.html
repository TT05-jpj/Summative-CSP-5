How to use My Objects

Add an object — Tap the yellow Add Object card to open the scanner. Point the camera at an object, tap Scan, then save it with a name and instructions.

View Instructions — See exactly what the user will read when they scan that object.

Edit — Update the name or instructions for an existing object.

Delete — Permanently remove an object from the list.

Languages — Use the dropdowns at the top to set the caretaker and user languages.

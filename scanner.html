How to use the Scanner

Enable camera — Tap Enable Camera and allow access when prompted.

Scan an object — Point the camera at the object and tap Scan Object. AI identifies it automatically.

Name and save — Edit the name, add instructions, then tap Save Object.

Mic buttons — Tap the mic icon to speak instead of type.

Done? — Tap Dashboard to return to your objects.

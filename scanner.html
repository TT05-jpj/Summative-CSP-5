How to use the Caretaker Scanner

Enable the camera — Click Enable Camera and allow access when prompted by your browser.

Scan an object — Point the camera at the object you want to add, then tap Scan Object. The AI will identify it automatically.

Name and save — Confirm or edit the object name, add instructions for the user, then tap Save Object.

Mic buttons — Tap the microphone icon next to any field to speak instead of type.

Done? — Click Dashboard to return to the object list.
